Buttons with Margin

In many (but not all) cases, buttons will need a margin. To add a default margin, use the gcls-btn--margin css class.

<button type="button" class="btn btn-primary gcls-btn gcls-btn--margin">Primary</button>
<button type="button" class="btn btn-primary  gcls-btn gcls-btn--margin">Primary</button>
